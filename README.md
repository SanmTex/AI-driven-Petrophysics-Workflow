# AI-driven Petrophysics Workflow

Interactive Jupyter notebook and supporting CSVs for ensemble-regression prediction of missing reservoir properties - Resistivity in this specific case.

Usage
1. Clone the repo and run Jupyter from the repo root:
   jupyter lab
2. Open the notebook and run cells top-to-bottom.
3. Data files (csv) are included in the repository root (or in `data/` if you move them).

Dependencies
See requirements.txt for Python packages.
